# 채널별 연결 가이드 설계

작성일: 2026-07-19

상태: 확정

선택안: 채널 카드별 상세 모달

## 목적

고객이 외부 개발자 문서를 해석하거나 계정 비밀번호와 토큰을 직접 전달하지 않고도 Brand Pilot 연결에 필요한 계정 상태와 OAuth 승인 순서를 이해하게 한다.

## 적용 채널

- Instagram
- Threads
- X
- LinkedIn
- YouTube
- TikTok

Webflow는 제거된 채널이므로 가이드에 포함하지 않는다.

## 화면 구조

각 채널 카드에 `연결 가이드` 버튼을 표시한다. 버튼을 누르면 해당 채널만 설명하는 모달을 연다. 카드 안에는 연결 상태와 주요 동작만 유지하고 긴 설명은 넣지 않는다.

모달은 다음 순서로 구성한다.

1. 현재 지원 상태
2. 연결 전 준비 사항
3. 계정 유형 설정 방법
4. Brand Pilot OAuth 연결 순서
5. 요청 권한과 사용 목적
6. 연결 완료 확인 항목
7. 자주 발생하는 문제와 해결 방법
8. 플랫폼 공식 문서

## 공통 원칙

- 고객에게 플랫폼 아이디, 비밀번호, 액세스 토큰, 앱 시크릿을 입력받지 않는다.
- 고객은 플랫폼이 제공하는 OAuth 동의 화면에서 권한을 승인한다.
- 개발자 앱 생성, 앱 시크릿, Redirect URI, 앱 심사는 Brand Pilot 운영자 책임으로 표시한다.
- OAuth가 아직 구현되지 않은 채널도 가이드는 제공하되 `연결 준비 중` 상태를 명확히 표시한다.
- 공식 요구사항이 변경될 수 있으므로 각 가이드에 공식 문서 링크와 최종 확인 안내를 둔다.

## 채널별 핵심 범위

### Instagram

Instagram 전문 계정 중 비즈니스 계정 전환 방법을 상세히 제공한다. Instagram 앱에서 `프로필 → 메뉴 → 설정 및 활동 → 계정 유형 및 도구 → 프로페셔널 계정으로 전환 → 비즈니스` 순서로 안내한다. Facebook Page 생성·연결, Page 접근 권한, 올바른 Facebook 계정으로 OAuth 승인하는 방법을 이어서 설명한다.

현재 Brand Pilot의 Facebook Login 방식에서는 연결된 Facebook Page와 Instagram 전문 계정을 조회할 수 있어야 한다. 계정이 보이지 않을 때는 개인 계정 여부, Page 연결 여부, Facebook 계정의 Page 권한을 순서대로 확인한다.

### Threads

Threads 프로필 생성, 연결에 사용할 Meta 계정 확인, Threads OAuth 권한 승인 순서를 안내한다. 고객은 별도 Meta 개발자 앱을 만들지 않는다.

### X

게시 가능한 X 계정, OAuth 동의 계정 확인, 글 읽기·작성과 장기 연결 권한의 목적을 안내한다. Brand Pilot의 X 연결 기능은 준비 상태를 별도로 표시한다.

### LinkedIn

개인 프로필과 회사 Page 게시의 차이를 설명한다. 회사 Page로 게시하려면 해당 Page의 관리 권한이 필요함을 안내하고, OAuth에서 선택한 계정과 게시 대상을 확인하도록 한다.

### YouTube

Google 계정만 있고 YouTube 채널이 없는 경우 채널을 먼저 생성하도록 안내한다. 여러 채널이나 브랜드 계정이 연결된 경우 OAuth 화면에서 실제 게시할 채널을 선택하고 업로드 권한을 승인하도록 한다.

### TikTok

TikTok 계정 준비, Content Posting 권한 승인, 게시 공개 범위 확인을 안내한다. 심사되지 않은 앱에서는 게시 공개 범위가 제한될 수 있다는 플랫폼 제약도 표시한다.

## 접근성

- 모달은 `role="dialog"`, `aria-modal="true"`, 제목 연결을 사용한다.
- 열릴 때 닫기 버튼으로 포커스를 이동하고 닫을 때 원래 버튼으로 복귀한다.
- `Escape`와 배경 클릭으로 닫을 수 있다.
- 모달 내부에서 Tab 포커스가 빠져나가지 않도록 한다.
- 본문이 길면 모달 본문만 스크롤한다.

## 테스트

- 6개 채널에 각각 `연결 가이드` 버튼이 표시된다.
- Instagram 모달에 비즈니스 계정 전환, Facebook Page 연결, 비밀번호 미수집 안내가 표시된다.
- 각 채널 버튼은 해당 채널 제목과 준비 사항을 연다.
- 모달 닫기와 Escape 동작을 검증한다.
- 기존 OAuth 연결 버튼과 채널 활성화 동작은 변하지 않는다.
