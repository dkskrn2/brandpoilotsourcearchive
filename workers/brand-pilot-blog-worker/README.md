# 모종 Blog Worker

검색 의도 분석, 자연스러운 한국어 본문, 안전한 HTML과 1200x630 대표 PNG 1장을 생성한다. 본문 PNG는 비교·과정·구조·예시를 설명하는 데 꼭 필요한 경우에만 워커가 0~5장 범위에서 결정하며 장식용 이미지는 만들지 않는다. 생성된 본문 이미지는 `inline-01.png`부터 순서대로 HTML의 관련 위치에 모두 삽입되고, 구체적인 한국어 대체 텍스트를 가진다. 저장소 업로드 후 이미지 경로는 공개 URL로 치환된다. 중앙 `codex-cli/content` lease를 공유하며 블로그 작업을 한 번에 하나만 처리한다.
